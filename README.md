# WTTG3 - Traduction francaise

Pack debutant pour *Welcome to the Game III* (menus, UI, CryptChat, ACRS, PDF).

## Installation

1. Telecharge `WTTG3-FR-Traduction.zip` dans [Releases](../../releases)
2. Dezippe
3. Ferme le jeu
4. Lance `INSTALLER.bat`
5. Choisis le dossier du jeu (celui qui contient `WTTGSD\Content\Paks`)

## Desinstallation

Lance `DESINSTALLER.bat` (meme dossier de jeu).

Les sites Dark Net restent en anglais volontairement.
