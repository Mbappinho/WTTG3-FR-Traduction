WTTG3 - Traduction francaise (pack Nexus)
=========================================

COMPATIBILITE STEAM (v1.2.7)
- Jeu : Welcome to the Game III (Steam officiel)
- AppID : 3869850
- BuildID Steam cible : 24303741
- INSTALLER.bat DETECTE automatiquement ton BuildID et indique
  si c'est compatible (OK) ou non (risque de crash)
- Si BuildID different : telecharge une release FR a jour, ou
  n'installe pas (sinon crash possible)

CONTENU
- Menus / UI / CryptChat / ACRS / inventaire / DarkDrop (mod FR_P)
- Boutons patchables : ACHETER, Confirmer, Commander, Se cacher, etc.
- PDF in-game en francais (dont fiches Contestants)
- PAS de fichiers anglais d'origine redistribues (regles Nexus)

INSTALLER (premiere fois)
1. Ferme le jeu completement.
2. Double-clique sur INSTALLER.bat
3. Choisis le dossier du jeu (ou colle le chemin)
4. Confirme avec O
5. Relance le jeu

DESINSTALLER
1. Ferme le jeu.
2. Double-clique sur DESINSTALLER.bat (retire le mod FR_P)
3. Pour remettre les PDF en anglais :
   Steam > clic droit sur le jeu > Proprietes > Fichiers installes
   > Verifier l'integrite des fichiers

SI LE JEU CRASH AU LANCEMENT
1. DESINSTALLER.bat tout de suite (ou supprime WTTGSD\Content\Paks\WTTGSD-Windows_FR_P.*)
2. Verifie l'integrite Steam si besoin
3. Confirme que le jeu sans mod demarre
4. Installe uniquement une release FR rebuildée pour ta maj Steam

OU TROUVER LE DOSSIER DU JEU
Steam : clic droit sur le jeu > Gerer > Parcourir les fichiers locaux
Le dossier utile contient WTTGSD\Content\Paks

NOTES
- Fan translation non officielle, non affiliee a Reflect Studios
- Les sites web Dark Net restent en anglais (volontaire)
- Si Windows bloque le .bat : "Plus d'infos" > "Executer quand meme"
- Source / mises a jour : https://github.com/Mbappinho/WTTG3-FR-Traduction

Garde ce dossier entier. Ne separe pas INSTALLER.bat du dossier fichiers\
