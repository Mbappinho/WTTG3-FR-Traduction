WTTG3 - Traduction francaise (pack GitHub Full)
===============================================

COMPATIBILITE STEAM (v1.5.0)
- Jeu : Welcome to the Game III (Steam officiel)
- AppID : 3869850
- BuildID Steam cible : 24359942
- Fichier fichiers\steam_target.json = version du pack + BuildID cible
- INSTALLER.bat DETECTE ton BuildID Steam (OK / ATTENTION crash)
- INSTALLER.bat peut TELECHARGER la derniere release GitHub
  (version plus recente ou mieux adaptee a ton BuildID)
  → demande O/N ; internet requis
- Si BuildID different et pas de maj GitHub : n'installe PAS
  (sinon crash possible au lancement)

CONTENU
- Menus / UI / CryptChat / ACRS / inventaire / DarkDrop (mod FR_P)
- PDF in-game FR (Contestants, Hacks, Threats, VirtMesh, etc.)
- Achievements FR si le dossier steam_settings existe
- Auto-update GitHub via INSTALLER.bat (pack Full uniquement)

------------------------------------------------
INSTALLER (premiere fois)
------------------------------------------------
1. Ferme le jeu completement.
2. Dezippe CE pack quelque part (garde le dossier entier).
3. Double-clique sur INSTALLER.bat
4. Indique le dossier du jeu si non detecte
   (Steam > clic droit > Gerer > Parcourir les fichiers locaux)
   Exemples :
     C:\Program Files (x86)\Steam\steamapps\common\Welcome to the Game III
     C:\Steam\steamapps\common\Welcome to the Game III
5. Si l'installeur propose une maj GitHub plus recente : reponds O
6. Confirme l'installation avec O
7. Relance le jeu

------------------------------------------------
METTRE A JOUR LE MOD (proprement)
------------------------------------------------

Methode A — recommandee (auto-update)
Disponible UNIQUEMENT si ton pack a deja INSTALLER.bat avec maj GitHub
(= releases Full **v1.4.1** et **v1.4.2** ; les packs plus vieux n'ont PAS l'auto-update).
1. Ferme le jeu.
2. Relance INSTALLER.bat de TON pack actuel (1.4.1 ou 1.4.2).
3. Quand il affiche "Une version plus recente..." → reponds O
4. Confirme l'installation du pack telecharge → O
5. Relance le jeu
Apres une maj Steam : si BuildID OK / maj GitHub proposee, Methode A suffit
souvent (pas besoin de DESINSTALLER avant).
Packs sans INSTALLER.bat (Nexus drop-in, ou Full avant 1.4.1) → Methode B.

Methode B — manuelle (nouveau zip telecharge a la main)
1. Ferme le jeu.
2. Telecharge la DERNIERE release Full :
   https://github.com/Mbappinho/WTTG3-FR-Traduction/releases
   Asset : WTTG3-FR-Traduction.zip
3. DESINSTALLER.bat (ancien pack) — propre
4. INSTALLER.bat du NOUVEAU zip
5. Relance le jeu

Apres une MAJ Steam du jeu :
- Si l'installeur dit BuildID OK → Methode A suffit souvent
- Si CRASH au lancement → DESINSTALLER tout de suite, puis
  installe UNIQUEMENT une release FR rebuildée pour le nouveau BuildID
  (un vieux zip ne suffit en general PAS)
- Ne colle pas un vieux FR_P par-dessus apres une maj Steam

------------------------------------------------
DESINSTALLER
------------------------------------------------
1. Ferme le jeu.
2. DESINSTALLER.bat
3. Meme dossier de jeu → confirme O
(ou supprime WTTGSD\Content\Paks\WTTGSD-Windows_FR_P.*)

------------------------------------------------
SI LE JEU CRASH AU LANCEMENT
------------------------------------------------
1. DESINSTALLER.bat immediatement
2. Verifie que le jeu SANS mod demarre
3. Installe la derniere release FR compatible (BuildID)
4. Si aucune release a jour : joue sans mod en attendant

------------------------------------------------
NOTES
------------------------------------------------
- Sites web Dark Net = anglais (volontaire)
- Windows bloque le .bat → "Plus d'infos" > "Executer quand meme"
- Erreur "jeu ouvert" → ferme WTTGSD dans le Gestionnaire des taches
- Pack Nexus (sans .bat) = autre zip, drop-in manuel, PAS d'auto-update
- steam_target.json sur la release GitHub = meta legere pour l'auto-update
  (ne remplace pas le zip Full)
